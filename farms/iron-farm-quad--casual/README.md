# Iron Farm - Quad

Uses villager panic mechanics to summon Iron Golems to collect iron.

![Build Preview](./images/preview.png)

## Minecraft Compatibility

**Edition:** Java  
**Tested Version(s):** 1.21  
**Requires Mods:** No

## Build Details

**AFK Required:** No, but the farm must be in loaded chunks to work.  
**Notes or Tips:** Zombie must be tagged or holding an object to prevent despawning when unloaded.
