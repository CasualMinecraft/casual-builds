## 🗂 Build Categories

Current build categories include:

- [**Bases**](./bases/): Primary living areas, starter homes, mega bases, or survival hubs.
  - Focus: Livability, organization, expansion potential.
  - Examples: Starter Survival House, Underground Bunker Base, Mountain Cliffside Base, Ocean Monument Mega Base, Hardcore Fortress Base, Compact Storage-Centered Base
- [**Farms**](./farms/): Resource-generating builds designed for automation and efficiency.
  - Focus: Rates per hour, AFK potential, survival practicality.
  - Examples: Iron Farm, Creeper Gunpowder Farm, Gold Farm, Wither Skeleton Farm, Sugar Cane Farm, Villager Breeder

- [**Redstone**](./redstone/): Mechanics-focused builds centered around circuitry and automation.
  - Focus: Circuit design, compactness, innovation.
  - Examples: 3x3 Piston Door, Hidden Staircase Entrance, Automatic Smelter, Item Sorter System, Redstone Elevator, Flying Machine Door

- [**Decorative**](./decorative/): Aesthetic builds focused primarily on visual appeal.
  - Focus: Block palette, style, architecture, detailing.
  - Examples: Medieval Castle, Fantasy Tower, Custom Nether Portal Frame, Statue or Monument, Custom Tree Design, Detailed Interior Build

- [**Utilities**](./utilities/): Quality-of-life builds that improve world functionality but aren’t strictly farms or bases.
  - Focus: Convenience, organization, accessibility.
  - Examples: Mega Storage Hall, Shulker Box Unloader, Map Wall System, Villager Trading Hall, Nether Portal Hub, Enchanting Setup Room

### Inside Each Build Folder

- schematics/
- images/
- A `README.md` file containing:
  - At least one image preview of the build
  - A complete material list for each included schematic
  - Minecraft version compatibility (e.g., Java 1.21+)
  - Creator information

## 🧱 How To Use Schematics

### WorldEdit

Place `.schem` files inside your WorldEdit schematics folder. Load and place in-game using the following commands:

```
//schem load [build-name]
//paste
```

### Litematica

Place `.litematic` files inside your Litematica schematics folder. Load in-game using the Litematica menu (default `M`).
